# Node.js AI App

Next.js를 사용한 웹 애플리케이션입니다.

## 기술 스택

- Next.js
- TypeScript
- Tailwind CSS
- ESLint

## 프로젝트 구조

```
src/
├── app/                # 페이지 컴포넌트
├── components/         # 재사용 가능한 컴포넌트
└── styles/            # 스타일 파일
```

## 주요 기능

- 반응형 네비게이션 바
- 라우팅 기능이 있는 메뉴 (Top, History, Product, Etc)
- 각 페이지별 레이아웃

## 개발 환경 설정

1. 저장소 클론
```bash
git clone [repository-url]
```

2. 종속성 설치
```bash
npm install
```

3. 개발 서버 실행
```bash
npm run dev
```
